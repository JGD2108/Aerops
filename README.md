# AeroOps NoSQL Control Tower

AeroOps NoSQL Control Tower is a cloud-native operational data platform for aviation/travel workloads.

It models flights, operational flight events, airports, audit runs, and synthetic passenger itineraries using MongoDB-style NoSQL design.

The API supports operational access patterns such as delayed flight lookup, event timelines, impacted passenger queries, delay summaries, and audit run review.

## Business Context

This project is aligned with a Data Engineer – Operational Data (NoSQL) role in the aviation/travel technology sector.

The goal is to demonstrate NoSQL data modeling, indexing strategy, operational APIs, Docker-based reproducibility, Kubernetes deployment readiness, and an Azure-ready architecture.
